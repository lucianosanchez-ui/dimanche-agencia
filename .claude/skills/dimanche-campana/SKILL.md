# dimanche-campana

Motor: skill `campaign-plan` del plugin marketing. Anclaje: MAN-018 (campañas anuales), POL-007 (posicionamiento), POL-016 (KPIs), drivers.

## Proceso
1. Definí **objetivo ligado a un KPI POL-016** (seguidores hacia 10K, rating, tráfico, ventas digitales).
2. Pilar dominante (POL-015) + buyer persona + período + ángulo de marca.
3. Si es **pauta**, ver la receta de paid media (audiencias por persona, presupuesto 5-7% ingresos); el Estratega lo propone, **gate de Luciano antes de gastar**.
4. Listá las piezas que la componen (van al Calendario).

## Output
Item en la base *Objetivos & Campañas* con Tipo (Campaña anual / táctica / Promo / Publicidad), Estado **Propuesto**, objetivo/KPI, pilar, fechas. Nada se activa sin OK de Luciano. Respetar T2 (bajo volumen / alta calidad).
